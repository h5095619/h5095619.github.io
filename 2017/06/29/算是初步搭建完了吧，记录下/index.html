第一篇说下Hexo+Github搭建博客吧

发表于 2017-06-29 | 阅读次数

用了大概一天左右时间(不算备案),算是初步把博客搭建起来了,也是蛮开心的。之前跟一些朋友讨论，现在建博客是不是晚了,因为看很多大牛都是学生时就建自己的博客了，我想说的是并不晚，有一句话很好:种树的最佳时间是25年前,仅次于它的最佳时间是现在。
那么现在，我也来个手把手教你在Windows下搭建博客哟，哈哈哈

要准备的:

  1. GitHub (记住用户名和邮箱)
  2. 下载 Node.js和 Git
  3. 安装Hexo

要开始了

  • Github搭建个人仓库
  • 安装Git
  • 安装Node.js
  • 安装Hexo
  • 观赏下我们的Hexo
  • 托管到Github
  • 完成
搭建个人仓库

注册-登录—点击右上角新建仓库

填写Repository name 这里格式就填写自己的用户名.github.io这个之后就是你的访问域名了，创建成功页面如下:


Ps:红色箭头的地址我们可以复制一下建个txt暂时存下,在配置hexo时我们会用到

安装Git

Git是一款免费、开源的分布式版本控制系统，用于敏捷高效地处理任何或小或大的项目。

将我们下载的Git安装，详细的可以参考廖雪峰老师的Git教程。安装完成之后我们cmd打开命令行,输入git看到如下图所示就是安装成功了



安装成功后右击或者菜单里搜索会看见Git Bash。打开它，我们绑定github的用户名和邮箱,输入如下命令:

git config –global user.name “你的用户名”
git config –global user.email “你注册时的邮箱”
都是英文分号

安装Node.js

Node.js安装教程很详细,安装完后我们打开命令行输入指令验证下,如下图所示:

node –version



成功的话可以看到Node.js的版本号

安装Hexo

我们刚才安装了Node.js,这时候我们也同时安装了NPM.

NPM是随同NodeJS一起安装的包管理工具，能解决NodeJS代码部署上的很多问题.
输入如下命令行:
npm -v



成功看到版本号


好了好了开始安装Hexo，新建一个你放博客的文件夹比如Blog,然后鼠标右击选择在此处打开命令行窗口,
输入以下命令

npm install -g hexo-cli

这里有个坑就是我知道这里安装会很慢但，等了半天还没反应，原来是需要我敲下回车，于是就继续进行了。

观赏下我们的Hexo

到这里马上就可以看到我们的小成果了

安装完成后我们要初始化我们的博客，输入如下命令行:

hexo init blog
（还是在这个文件夹下,右击选择在此处打开命令行）

初始化完成后，你会看见在你的文件夹下多了好多新的文件夹哈，会提示
INFO Start blogging with Hexo!马上了，马上了我们就可以看见效果了,
输入命令行,如下图所示:

hexo new my_blog

hexo g

hexo s




明显可以看出让我们去访问这个地址哈，激不激动，激不激动？Go!

这是打开后的界面


我的用的NextT主题，当然还有各种主题，大家的应该是默认landscape主题




激动激动，惊不惊喜。当然了，还没有完全搞完哈

托管到Github

让更多的人看到你的博客,Github的影响力就不用说了，就冲免费，这服务也是很好的哈，开玩笑了哈。我们继续.

打开Git Bash输入如下命令

ssh-keygen -t rsa -C “你当时注册github的邮箱”
需要回车就回车然后最后会告诉你生成文件的路径,找到这个公钥文件(id_rsa.pub)，私钥自己留在电脑上，打开它，解开它的扣子哦不，打开它，全选复制所有内容。然后打开Github进入你刚才创建的仓库，选择Settings然后选择左侧Deploy keys然后点击Add deploy key,看图：



这个Title可以随便写，然后key填写你刚才复制的文件里的内容,最后点击Add key完成

好，我们回头修改下博客配置文件就差不多了，这里有一个知识点，我们的博客下有两个_config.yml文件，一个是最外层这个我们称作站点配置文件，另一个是theme主题下的我们称作主题配置文件。接下来我们修改站点配置文件，打开然后翻到最下面填写一些个配置，如下所示：



其中repo的值填写我们最刚开始让你复制保存的你的github仓库地址，如果没有保留就进入github复制你的仓库地址，最后加上.git即可，记住这中间都是有空格的，至于为什么就不做解释了，之后的markdown书写习惯都是这样的。这里还有个小坑，要配置下，否则我们的博客会出现乱码，翻到最上面，如图所示:



设置下语言，然后我们保存的时候要保存成UTF-8格式,好了我们开始发布。
在blog目录下,shift+右击在此处打开命令行，输入如下命令:


hexo clean

hexo g

hexo d







Ps:第一次上传到Github时可能会弹框让你输入帐号密码，之后就不会了因为我们已经配置了SSH。

完成

如果没什么问题，我们打开浏览器，输入自己的仓库地址就可以看到我们自己的博客了哈哈，这下激不激动，惊不惊喜哈。
